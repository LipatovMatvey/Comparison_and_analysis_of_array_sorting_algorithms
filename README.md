# Документация проекта

Данный проект предназначен для сравнения и анализа алгоритмов сортировки массивов. Включает в себя реализацию следующих алгоритмов:

1. Сортировка пузырьком
2. Быстрая сортировка
3. Сортировка слиянием

## Установка

Чтобы запустить проект, вы можете склонировать его с репозитория:

```bash
git clone https://github.com/LipatovMatvey/Comparison_and_analysis_of_array_sorting_algorithms.git
```

Далее перейдите в папку с проектом и установите необходимые зависимости.

## Как использовать

Для использования проекта просто запустите главный файл:

```bash
python main.py
```

## Примеры

В проекте содержатся примеры использования различных алгоритмов сортировки с различными входными данными.
