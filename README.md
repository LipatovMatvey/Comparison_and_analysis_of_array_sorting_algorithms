# 🔄 Сравнение и анализ алгоритмов сортировки массивов

[![C#](https://img.shields.io/badge/Language-C%23-blue)](https://docs.microsoft.com/en-us/dotnet/csharp/)
[![.NET](https://img.shields.io/badge/.NET-Framework%204.5%2B-blueviolet)](https://dotnet.microsoft.com/download)

> Windows-приложение для анализа и сравнения производительности алгоритмов сортировки

## 📋 Описание

Интерактивное приложение для тестирования и визуализации производительности трёх алгоритмов сортировки:
- **Heap Sort** - O(n log n)
- **Shell Sort** - O(n log² n)  
- **Counting Sort** - O(n + k)

## 🎯 Функции

- ✅ Выбор размера массива
- ✅ 4 типа упорядоченности данных
- ✅ Таблица результатов (время в миллисекундах)
- ✅ Интерактивный график сравнения
- ✅ 30 повторений для точности

## 🛠️ Технологии

- **Язык**: C#
- **Фреймворк**: .NET Framework 4.5+

- **Визуализация**: Chart Control

## 📖 Алгоритмы

### Heap Sort (Пирамидальная сортировка)
- Сложность: O(n log n) во всех случаях
- Память: O(1)
- Построение кучи и последовательное извлечение элементов

### Shell Sort (Сортировка Шелла)
- Сложность: O(n log² n) с последовательностью Седжвика
- Память: O(1)
- Обобщение сортировки вставками с уменьшающимися промежутками

### Counting Sort (Сортировка подсчётом)
- Сложность: O(n + k)
- Память: O(n + k)
- Линейная сортировка для целых чисел

## 🚀 Установка

1. Откройте Visual Studio
2. Откройте `Lipatov_Matvey_24VP2_KR.sln`
3. Скомпилируйте (Ctrl+Shift+B)
4. Запустите (F5)

## 📊 Использование

1. Введите размер массива
2. Выберите тип упорядоченности
3. Нажмите "Сортировать"
4. Просмотрите результаты в таблице и графике

## 📈 Сравнение

| Алгоритм | O(лучший) | O(средний) | O(худший) |
|----------|-----------|-----------|----------|
| Heap Sort | n log n | n log n | n log n |
| Shell Sort | n log n | n log² n | n log² n |
| Counting Sort | n+k | n+k | n+k |

## ⚠️ Ошибки

Приложение выводит ошибку красного цвета при:
- Неверном размере массива
- Незыбранном типе упорядоченности
- Неверном проценте упорядоченности
